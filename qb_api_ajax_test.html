About This Page:

This page was created as a native QuickBase alternative to Postman. The intent is to provide a user with the ability to easily select and adjust parameters to automatically generate an AJAX API call and provide a response. The User can then copy and paste this request straight into his or her code.

This page uses JSON inputs which it then converts into XML. This is to better align with QuickBase's upcoming changes to it's API.

Future improvements include allowing the user to select XML or JSON as a response.
